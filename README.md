# Portfolio - Chittibala Teja Nandiswar Reddy

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🏠 **Home Page** - Professional introduction with profile photo and social links
- 💼 **Projects** - Showcase of completed projects with descriptions and tech stack
- 📝 **Experience** - Timeline of internships and experience with skills gained
- 📄 **Resume** - Resume preview and download option
- 📧 **Contact** - Contact form and social media links
- 📱 **Responsive** - Fully responsive design for all devices
- ✨ **Smooth Animations** - Engaging animations and transitions

## Project Structure

```
portfolio/

```

## Getting Started

### Prerequisites
- A modern web browser
- Text editor (VS Code, Sublime Text, etc.)
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/NandhuTeja/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser:
```bash
open index.html
# or simply double-click the file
```

## Customization

### Update Personal Information

1. **Profile Photo**: Replace the placeholder image URL
   ```html
   <img src="path/to/your/photo.jpg" alt="Profile Photo">
   ```

2. **Contact Information**:
   - Update email address
   - Update LinkedIn profile URL
   - Update GitHub profile URL

3. **Projects**: Modify project cards in the Projects section with:
   - Project title
   - Description
   - Technologies used
   - GitHub and demo links

4. **Experience**: Update internship details:
   - Company names
   - Dates
   - Responsibilities
   - Skills gained

5. **Resume**: Replace the resume PDF path with your actual resume

## Deployment

### Deploy on GitHub Pages

1. Ensure repository is named `portfolio`
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Your portfolio will be live at `https://yourusername.github.io/portfolio`

### Deploy on Other Platforms

- **Netlify**: Connect GitHub repo and auto-deploy
- **Vercel**: Similar process as Netlify
- **Firebase**: Follow Firebase hosting guide
